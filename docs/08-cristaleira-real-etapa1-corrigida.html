1.3. Desafios Específicos Identificados
Vedação 3 Portas
  • Problema: 3 portas = múltiplas junções de vedação
  • Solução: Sistema vedação escalonada nas sobreposições
  • Material: EVA em todas junções + cantos reforçados
  • Teste: Verificação independente cada porta
Pontes Térmicas Múltiplas
  • Dobradiças externas: 6 pontos de condução térmica
  • Maçanetas/fechaduras: Metal condutor
  • Junções madeira: Continuidade térmica
  • Solução: Isolamento contínuo + quebra térmicas
Base com Gavetas
  • Vantagem: Isolamento do solo melhorado
  • Desafio: Acesso à parte inferior limitado
  • Solução: Isolamento superior da base interna
  • Coolers: Posicionamento otimizado acima gavetas

3. PROCEDIMENTO DE INSTALAÇÃO
PREPARAÇÃO (2h)
  1. Esvaziar completamente cristaleira
  2. Remover prateleiras/suportes internos existentes
  3. Limpeza rigorosa com álcool isopropílico
  4. Secar completamente (aguardar 1h)
  5. Conferir dimensões internas exatas:
    • Altura: 157 cm
    • Largura: 175 cm
    • Profundidade: 49 cm
  6. Identificar e marcar pontes térmicas (dobradiças, fechaduras)
  7. Avaliar condição das gavetas (vedação base)
ETAPA 1: PAREDE TRASEIRA (3h)
  1. Cortar manta térmica: 180cm × 180cm
  2. Aplicar fita dupla face nas bordas parede externa
  3. Fixar 1ª camada (face metalizada FORA)
  4. Pressionar eliminando bolhas de ar
  5. Marcar grid espaçadores: 40×40cm (20 pontos total)
  6. Colar espaçadores 10×10×10mm com cola instantânea
  7. Aguardar cura 30min
  8. Fixar painel MDF 5mm com parafusos autorroscantes
  9. Cortar 2ª camada: 174cm × 174cm
  10. Fixar 2ª camada no MDF (face metalizada DENTRO)
ETAPA 2: LATERAIS (2h)
  1. Lateral esquerda: 180cm × 49cm
  2. Lateral direita: 180cm × 49cm
  3. Sistema dupla camada com espaçadores
  4. Sobreposição 2cm com manta traseira
  5. Atenção às dobradiças: isolamento específico
ETAPA 3: TOPO (2h)
  1. Cortar manta: 180cm × 49cm
  2. Sistema dupla camada com espaçamento 10mm
  3. Região crítica: acúmulo calor
  4. Preparar pontos furos exhaust coolers
  5. Vedação cuidadosa cantos superiores
ETAPA 4: BASE ESPECIAL (3h)
  1. Base sobre gavetas: isolamento superior
  2. Espaçamento 15mm (vs. 10mm demais paredes)
  3. Razão: acomodar fiação coolers
  4. Cortar manta: 180cm × 49cm
  5. Grid espaçadores: evitar área dos 4 coolers
  6. Marcar posições intake (sem furar ainda)
  7. Reforço isolamento sobre gavetas metálicas
ETAPA 5: VEDAÇÃO TRIPLA PORTAS (2h)
  1. Aplicar fita EVA 6mm em todas junções:
    • Junção porta A/parede lateral esquerda
    • Junção porta A/porta B (sobreposição)
    • Junção porta B/porta C (sobreposição)
    • Junção porta C/parede lateral direita
    • Junções todas portas com topo
    • Junções todas portas com base
  2. Teste vedação porta por porta
  3. Teste conjunto: todas portas fechadas
  4. Reforço pontos vazamento detectados

4.2. Vantagens Sistema Unificado
Circulação Otimizada
  • Zona superior livre: 140cm altura para circulação
  • Fileira única: Instrumentos protegidos na base
  • Acesso facilitado: 3 portas = manutenção fácil
  • Fluxo otimizado: Ar circula acima dos instrumentos
Eficiência Energética
  • 4 coolers vs. 6 originais: 33% menor consumo
  • Volume único: Menos perdas térmicas
  • Controle centralizado: Otimização automática
  • Base isolada: Menor ganho térmico solo
