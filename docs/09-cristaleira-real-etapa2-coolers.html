1.3. Princípio de Funcionamento
Proteção Instrumentos
  • Intake protegido: Defletores EVA direcionam ar para cima
  • Zona estável: Instrumentos na região de menor turbulência
  • Gradiente suave: Variação temperatura <1°C na base
  • Fluxo indireto: Ar nunca incide diretamente nos instrumentos
Circulação Otimizada
  • Convecção forçada: Ar quente sobe naturalmente auxiliado pelos exhaust
  • Renovação controlada: 2-3 trocas/hora (vs. 8-10 sistemas convencionais)
  • Pressão equilibrada: Intake ≈ Exhaust (sem correntes fortes)
  • Distribuição uniforme: 175cm largura coberta pelos 4 coolers

3. PROCEDIMENTO DE INSTALAÇÃO
⚠️ CUIDADOS CRÍTICOS

INSTRUMENTOS: Remover TODOS os instrumentos antes da furação

PROTEÇÃO: Cobrir interior com lona durante trabalhos

POEIRA: Aspirar completamente após cada etapa

VIBRAÇÃO: Usar furadeira com controle de torque

PREPARAÇÃO (1h)
  1. Verificar conclusão Etapa 1 e cura 48h
  2. Remover todos instrumentos e proteger interior
  3. Conferir coordenadas marcadas para coolers
  4. Testar componentes elétricos antes instalação
  5. Organizar ferramentas e materiais
ETAPA 1: FURAÇÃO CONTROLADA (2h)
  1. Marcar centros exatos dos 4 furos Ø10mm nas coordenadas
  2. Furação gradual para evitar estilhaçamento:
    • Furo piloto Ø3mm em todas camadas
    • Expandir Ø6mm (manta externa + câmara ar)
    • Expandir Ø8mm (incluindo MDF)
    • Finalizar Ø10mm (manta interna)
  3. Lima circular para acabamento perfeito
  4. Aspirar poeira completamente após cada furo
  5. Teste passagem ar: soprar cada furo individualmente
ETAPA 2: MONTAGEM INTAKE BASE (C1, C2) (2h)
  1. Instalar filtros nos coolers C1 e C2
  2. Posicionar coolers sobre furos base (Z=170mm)
  3. Verificar orientação: fluxo PARA CIMA
  4. Fixar com parafusos M4×40mm e arruelas
  5. Instalar defletores EVA 45° sobre cada cooler:
    • Cortar EVA 5mm em formato cônico
    • Fixar com cola quente resistente
    • Teste direcionamento: ar deve ir lateralmente
  6. Vedação perímetro com espuma
  7. Conectar cabos (extensões se necessário)
ETAPA 3: MONTAGEM EXHAUST TOPO (C3, C4) (2h)
  1. Acesso pelo topo da cristaleira
  2. Posicionar coolers C3 e C4 sobre furos (Z=1550mm)
  3. Verificar orientação: fluxo PARA FORA (exhaust)
  4. Fixação com parafusos M4×30mm
  5. Vedação crítica: ar quente escapa facilmente
  6. Aplicar silicone neutro ao redor se necessário
  7. Teste rotação livre manual
ETAPA 4: SISTEMA ELÉTRICO E CONTROLE (3h)
  1. Instalar controlador PWM duplo posição acessível
  2. Conectar C1+C2 (intake) no canal 1
  3. Conectar C3+C4 (exhaust) no canal 2
  4. Passar fiação pela câmara 15mm da base
  5. Saída externa via furo lateral (similar Etapa 1)
  6. Conectar fonte 12V/2A externa
  7. Vedar passagem cabo com silicone + ilhós
  8. Teste funcional: cada canal independente

ETAPA 5: INSTALAÇÃO SENSORES (1h)
  1. Sensor 1: Base próximo instrumentos (Z=100mm, centro)
  2. Sensor 2: Meio altura zona livre (Z=800mm, centro)
  3. Sensor 3: Topo zona livre (Z=1400mm, centro)
  4. Fixação discreta com fita dupla face
  5. Cabos organizados junto à lateral
  6. Display/interface em posição visível externa

4.2. Protocolo de Testes
Teste 1: Proteção Instrumentos (30min)
  1. Posicionar papel tissue próximo aos instrumentos
  2. Ligar sistema velocidade 100%
  3. Verificar ausência movimento direto do papel
  4. Confirmar defletores direcionando ar corretamente
  5. Ajustar ângulo defletores se necessário
Teste 2: Homogeneidade Térmica (2h)
  1. Monitorar 3 sensores simultaneamente
  2. Registrar variações por 2 horas
  3. Meta: diferença máxima 1°C entre sensores
  4. Ajustar velocidades se desvio detectado
  5. Teste com portas fechadas e abertas
Teste 3: Eficiência Energética (24h)
  1. Medição consumo real vs. teórico
  2. Teste diferentes velocidades
  3. Encontrar ponto ótimo eficiência/resultado
  4. Configurar timers para economia noturna
