4. INSTRUÇÕES DE FABRICAÇÃO
4.1. Preparação das Peças
CORTE DAS PEÇAS
  1. Cortar peças conforme dimensões exatas da tabela
  2. Utilizar serra esquadria ou circular com guia
  3. Acabamento bordas com lixa 120 → 220
  4. Verificar esquadro e dimensões finais
  5. Identificar cada peça com etiqueta
FABRICAÇÃO ESPAÇADORES
  1. Cortar ripas 10×10mm das sobras
  2. Cortar cubos 10mm de comprimento (120 unidades)
  3. Cortar cubos 15mm para base (20 unidades)
  4. Lixar faces para uniformidade
  5. Verificar dimensões com paquímetro
4.2. Pré-Montagem e Verificação
TESTE MONTAGEM À SECO
  1. Montar painéis temporariamente com sargentos
  2. Verificar encaixes e esquadros
  3. Confirmar dimensões finais montadas
  4. Marcar pontos ajustes se necessário
  5. Desmontar para entrega separada
IMPORTANTE:

NÃO MONTAR DEFINITIVAMENTE: Painéis serão instalados após isolamento

ENTREGA: Peças separadas + espaçadores + parafusos

COORDENAÇÃO: Marceneiro deve acompanhar marcação furos coolers

5. COORDENAÇÃO COM INSTALAÇÃO
5.1. Sequência de Trabalho
Etapa Responsável Prazo Entregáveis
1. Fabricação MDF Marceneiro 3-5 dias Peças cortadas + espaçadores
2. Isolamento térmico Instalador 2-3 dias Sistema isolamento + posição espaçadores
3. Instalação MDF Marceneiro + Instalador 1 dia Painéis fixados + furos coolers
4. Sistema coolers Instalador 1-2 dias Sistema completo funcional
5.2. Pontos de Coordenação
Marcação Furos Coolers
  • Quando: Durante instalação dos painéis MDF
  • Quantidade: 4 furos Ø10mm
  • Posições: 2 na base + 2 no topo
  • Responsável: Instalador marca, marceneiro executa
Fixação Espaçadores
  • Grid: 40×40cm distância entre espaçadores
  • Colagem: Cola madeira + pressão 30s
  • Cura: 30min antes fixar painéis
  • Verificação: Esquadro e nivelamento
Apoios Instrumentos
  • Localização: Base painel MDF
  • Posições: 10 apoios espaçados 175mm
  • Material apoio: Fornecido pelo instalador (EVA)
  • Marcação: Durante instalação final
