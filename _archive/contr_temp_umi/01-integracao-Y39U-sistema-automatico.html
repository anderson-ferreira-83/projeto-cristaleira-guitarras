4. GUIA DE INSTALAÇÃO
4.1. Etapa 1 - Preparação (2-3 horas)
PREPARAÇÃO

Materiais e Ferramentas

Ferramentas necessárias:

  • Furadeira com broca 8mm e 25mm
  • Alicate de corte e desencapador
  • Chaves de fenda/phillips
  • Multímetro digital
  • Fita isolante e abraçadeiras

Verificações iniciais:

  1. Confirmar voltagem da rede (110V ou 220V)
  2. Verificar se Y39U está configurado de fábrica
  3. Testar sensores com multímetro (NTC ≈ 10kΩ a 25°C)
  4. Conferir todos os materiais da lista
4.2. Etapa 2 - Montagem Caixa de Controle (3-4 horas)
MONTAGEM ELÉTRICA
  1. Preparar caixa hermética:
    • Fazer recorte 67,5×25,5mm para Y39U
    • Furos para prensa-cabos (entrada sensores e saídas)
    • Furo para disjuntor externo
  2. Instalar Y39U na caixa:
    • Encaixar pela frente no recorte
    • Fixar com braçadeiras laterais incluídas
    • Verificar vedação IP67
  3. Conectar alimentação:
    • Entrada: cabo PP 2×1,5mm² da rede
    • Terminal L: fase (fio marrom/preto)
    • Terminal N: neutro (fio azul)
    • Aterramento na caixa metálica
  4. Conectar saídas (relés):
    • OUT1 (C1→NO1): para tomada desumidificadores
    • OUT2 (C2→NO2): para tomada fonte coolers
    • OUT3: deixar livre (reserva)
  5. Organizar cabos internos:
    • Usar canaletas ou abraçadeiras
    • Identificar cabos com etiquetas
    • Deixar folga para manutenção
4.3. Etapa 3 - Instalação Sensores (2 horas)
POSICIONAMENTO SENSORES
  1. Sensor 1 - Temperatura Superior (P1):
    • Altura: 145cm do piso (zona superior)
    • Posição: lateral esquerda, 20cm da traseira
    • Fixação: suporte plástico ou fita dupla-face
    • Cabo: passar por canaleta até Y39U (2m)
    • Polaridade: não importa para NTC
  2. Sensor 2 - Umidade Centro (P2):
    • Altura: 80cm do piso (zona central)
    • Posição: centro, longe de correntes de ar
    • Fixação: suporte específico do sensor
    • Cabo: passar por canaleta até Y39U (1,5m)
    • Polaridade: verificar manual (geralmente 3 fios)
  3. Sensor 3 - Temperatura Inferior (P3) - OPCIONAL:
    • Altura: 15cm do piso (zona inferior)
    • Posição: lateral direita, 20cm da traseira
    • Fixação: suporte plástico
    • Cabo: passar por canaleta até Y39U (1m)
ATENÇÃO - POSICIONAMENTO CRÍTICO DOS SENSORES:
  • NÃO posicionar sensores próximo aos coolers (leitura distorcida)
  • NÃO deixar sensores tocando instrumentos (falsa leitura)
  • NÃO colocar sensor umidade muito perto da parede (condensação)
  • SIM proteger cabos dos sensores (não tracionar)
  • SIM testar leitura antes de fixar permanentemente
4.4. Etapa 4 - Conexão dos Dispositivos (1 hora)
LIGAÇÃO DISPOSITIVOS
  1. Desumidificadores:
    • Plugar ambos na tomada OUT1
    • Se usar 2 desumidif.: ligar em paralelo (benjamim)
    • Ligar chaves dos desumidificadores em ON (sempre)
    • Controle será feito pelo relé do Y39U
  2. Coolers:
    • Fonte 12V 5A plugada na tomada OUT2
    • Ligar os 4 coolers na fonte (em paralelo)
    • Não precisa interruptor na fonte
    • Y39U liga/desliga fonte conforme temperatura
  3. Teste inicial (sem energia):
    • Conferir todas conexões com multímetro
    • Verificar curto-circuito (ohmímetro)
    • Confirmar polaridade dos sensores
    • Revisar checklist de instalação

5.2. Sequência de Programação
PROGRAMAÇÃO PASSO A PASSO
  1. Entrar no menu: Pressionar SET por 3 segundos
  2. Navegar: Usar ▲/▼ para mudar parâmetro, SET para confirmar
  3. Programar umidade:
    • HYS = 50 (setpoint 50% UR)
    • HYH = 3 (histerese 3%)
    • HAH = 60 (alarme alto 60%)
    • HAL = 40 (alarme baixo 40%)
  4. Programar temperatura:
    • TSE = 23 (setpoint 23°C)
    • HYT = 1 (histerese 1°C)
    • TAH = 30 (alarme alto 30°C)
    • TAL = 16 (alarme baixo 16°C)
  5. Configurar saídas:
    • O1F = dh (desumidificador)
    • O2F = FA (ventilador/coolers)
    • O3F = OFF
  6. Sair do menu: Pressionar SET por 3 segundos ou aguardar timeout
  7. Verificar display: Deve mostrar temperatura/umidade alternadamente

5.3. Teste e Comissionamento
TESTE INICIAL
  1. Ligar sistema vazio (sem instrumentos):
    • Fechar cristaleira
    • Ligar disjuntor principal
    • Y39U deve ligar e mostrar leituras
    • Aguardar 10min para estabilização
  2. Verificar leituras dos sensores:
    • Temperatura: deve estar próxima de ambiente (20-25°C)
    • Umidade: deve estar na faixa local (40-70% UR)
    • Se diferença > 5°C ou > 10% UR: verificar sensores
  3. Testar saída 1 (desumidificadores):
    • Aumentar setpoint HYS acima da UR atual
    • Relé deve fechar e ligar desumidificadores
    • Ouvir clique do relé e verificar LED OUT1
    • Retornar HYS para 50%
  4. Testar saída 2 (coolers):
    • Reduzir setpoint TSE abaixo da temp atual
    • Relé deve fechar e ligar coolers
    • Verificar LED OUT2 e rotação dos coolers
    • Retornar TSE para 23°C
  5. Monitoramento contínuo (72h):
    • Deixar sistema funcionando 3 dias
    • Anotar temp e umidade 3×/dia
    • Verificar se sistema mantém 45-55% UR e 20-26°C
    • Ajustar setpoints se necessário
  6. Instalação dos instrumentos:
    • Somente após sistema estável por 3 dias
    • Monitorar primeiras 2 semanas diariamente
    • Ajustes finos conforme necessário
6. OPERAÇÃO E MANUTENÇÃO
6.1. Operação Normal
O que o Y39U faz automaticamente:
  • Quando UR > 53%: Liga desumidificadores (OUT1)
  • Quando UR < 50%: Desliga desumidificadores
  • Quando Temp > 24°C: Liga coolers (OUT2)
  • Quando Temp < 23°C: Desliga coolers
  • Quando alarme: Sinal sonoro + LED piscando
6.2. Leitura do Display
Modo Display O que Mostra Interpretação
Normal alternado 23°C → 50% → 23°C → 50%... Sistema OK, valores dentro da faixa
Piscando + beep HAL ou HAH ou TAL ou TAH Alarme ativado (valor fora dos limites)
LED OUT1 aceso Saída 1 ativa Desumidificadores ligados
LED OUT2 aceso Saída 2 ativa Coolers ligados
Erro "Err" Erro de sensor Verificar conexão dos sensores
6.3. Manutenção Preventiva
Semanal
  • Verificar leituras no display (temp e UR)
  • Confirmar que não há alarmes ativos
  • Observar funcionamento dos coolers quando ligam
Mensal
  • Limpar sensores com pano seco (não usar água)
  • Verificar se relés estão acionando corretamente
  • Limpar coolers (pó acumulado nas pás)
  • Regenerar sílica dos desumidificadores
Trimestral
  • Conferir aperto de todos os terminais elétricos
  • Testar alarmes (forçar condições de alarme)
  • Verificar cabo dos sensores (não tracionar/romper)
  • Calibração de verificação (comparar com termômetro padrão)
6.4. Troubleshooting

Display mostra "Err" Sensor desconectado Verificar conexão P1, P2 ou P3
Leitura temp errada Sensor mal posicionado Afastar de correntes de ar/luz solar
UR não estabiliza Desumidif. não funciona Verificar OUT1 e desumidificadores
